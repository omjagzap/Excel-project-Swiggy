# 📊 Data Analysis & Dashboard Project

## 📌 Project Overview

Project 3 is developed using Microsoft Excel to perform data cleaning, transformation, analysis, and dashboard visualization.  
The purpose of this project is to convert raw data into meaningful business insights using analytical techniques and visual reporting.

---

## 🎯 Objectives

- Clean and preprocess raw dataset  
- Perform detailed data analysis  
- Identify key performance indicators (KPIs)  
- Generate insights using Pivot Tables  
- Create an interactive dashboard  

---


---

## 📊 Dataset Information

The Excel file includes the following sheets:

### 1️⃣ Raw Data
- Contains original dataset
- Transaction-level or record-level information
- Used for analysis and processing

### 2️⃣ Cleaned Data
- Removed duplicates  
- Handled missing values  
- Standardized formatting  
- Created calculated columns  

### 3️⃣ Analysis Sheet
- Pivot Tables  
- Summary calculations  
- Category-wise aggregation  

### 4️⃣ Dashboard Sheet
- KPI Cards  
- Bar Charts  
- Line Charts  
- Pie Charts  
- Interactive Slicers  

---

## 🧹 Data Cleaning Process

- Removed duplicate entries  
- Used `TRIM()` to remove extra spaces  
- Checked for blank or null values  
- Standardized date formats  
- Ensured correct numeric formatting  
- Created new calculated columns
